GNOME Desktop Environment
2017/04/23
 
Desktop Environment is not neccessary for Server usage, though. But Sometimes installation or using an application requires Desktop Environment, then build Desktop Environment as follwos.
[1] Install GNOME Desktop Environment.
root@dlp:~#
apt -y install gnome
# select Gdm3 or Lightdm during installation (this example selects Lightdm)
root@dlp:~#
reboot
[2] Click the icon which is located near the user name.
[3] Select [GNOME Classic] on this example. If you'd like to use Modern style's GNOME, select [GNOME], then, GNOME Shell starts.
[4] GNOME Desktop Environment starts.
