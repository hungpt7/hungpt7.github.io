WebMail system - SquirrelMail
  Install SquirrelMail to build web-based mail transfer system like yahoo mail or hotmail. SMTP/IMAP server is needed to run in your LAN.
[1] Install SquirrelMail
[root@www ~]#
yum -y install squirrelmail


[root@www ~]#
cd /usr/share/squirrelmail/plugins/

[root@www plugins]#
wget http://www.squirrelmail.org/plugins/compatibility-2.0.16-1.0.tar.gz

[root@www plugins]#
wget http://www.squirrelmail.org/plugins/empty_trash-1.4-1.2.2.tar.gz

[root@www plugins]#
wget http://www.squirrelmail.org/plugins/secure_login-1.4-1.2.8.tar.gz

[root@www plugins]#

[root@www plugins]#

[root@www plugins]#
