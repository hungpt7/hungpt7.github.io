Basic Operations
2014/12/25
 
Basic Operarions with virsh command
[1] Start Virtual Machine
[root@dlp ~]#
virsh start fedora21
# Start Virtual Machine 'fedora21'

Domain fedora21 started
[root@dlp ~]#
virsh start fedora21 --console
# start and connect to console of 'fedora21'

Domain fedora21 started
Connected to domain fedora21
[2] Stop Virtual Machine
[root@dlp ~]#
virsh shutdown fedora21
# Stop Virtual Machine 'fedora21'

Domain fedora21 is being shutdown
[root@dlp ~]#
virsh destroy fedora21
# Stop fourcely Virtual Machine 'fedora21'

Domain fedora21 destroyed
[3] Set auto-start for Virtual Machines
[root@dlp ~]#
virsh autostart fedora21
# Enable auto-start for 'fedora21'

Domain fedora21 marked as autostarted
[root@dlp ~]#
virsh autostart --disable fedora21
# Disable auto-start for 'fedora21'

Domain fedora21 unmarked as autostarted
[4] List all Virtual Machines
[root@dlp ~]#
virsh list
# List all active Virtual Machines

 Id    Name               State
----------------------------------------
 2     fedora21            running
[root@dlp ~]#
virsh list --all
# List all Virtual Machines included inactives

 Id    Name               State
----------------------------------------
 -     fedora21            shut off
 -     template           shut off
 -     Win2k12            shut off
[5] Switch console
Move to GuestOS to HostOS with Ctrl + ] key.
Move to HostOS to GuestOS with a command 'virsh console (name of virtual machine)'.
[root@dlp ~]#
virsh console fedora21
# connect to 'fedora21'

Connected to domain fedora21
Escape character is ^]
# Enter key
Fedora release 21 (Twenty One)
Kernel 3.17.4-301.fc21.x86_64 on an x86_64 (ttyS0)
localhost login:
# just switched on Guest
Password:
Last login: Sat Jul 12 20:51:10 2014
[root@localhost ~]#
# Ctrl + ] key

[root@dlp ~]#
# just switched on Host
