Management Tools
2015/11/22
 
Install useful tools for virtual machine management.
[1] Install required packages.
dlp:~ #
zypper -n install guestfs-tools libguestfs0 virt-top
[2] "ls" a directory in a virtual machine.
dlp:~ #
virt-ls -l -d sles12 /root 

total 32
drwx------ 1 root root    96 Nov 22 14:18 .
drwxr-xr-x 1 root root   156 Nov 22 14:09 ..
-rw------- 1 root root    28 Nov 22 14:20 .bash_history
drwx------ 1 root root    90 Nov 22 14:14 .gnupg
drwxr-xr-x 1 root root    10 Nov 22 14:18 .local
-rw-r--r-- 1 root root 28098 Nov 22 14:16 autoinst.xml
drwxr-xr-x 1 root root     0 Sep 21  2014 bin
drwxr-xr-x 1 root root    36 Nov 22 14:04 inst-sys
...
[3] "cat" a file in a virtual machine.
dlp:~ #
virt-cat -d sles12 /etc/passwd
