Services
2015/11/25
[1] It's possible to display all services like follows.
dlp:~ #
chkconfig --list | less
[2] Stop and turn OFF auto-start setting for a service if you don'd need it. (it's acpid as an example below)
dlp:~ #
/etc/init.d/acpid stop

Shutting down acpid     done
dlp:~ #
chkconfig acpid off
