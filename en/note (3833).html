Mail Log Analyzer - pflogsumm
2011/11/13
  Install pflogsumm that is Postfix mail log analyzer.
[root@mail ~]#
yum -y install postfix-perl-scripts
# generate log summary for yesterday

[root@mail ~]#
perl /usr/sbin/pflogsumm -d yesterday /var/log/maillog
